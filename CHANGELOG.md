# Histórico de mudanças

## 1.0.0

- Releitura do projeto original de terminal como aplicativo desktop para Windows em `C# + WPF`.
- Inclusão de operações científicas, números complexos, alternância entre graus e radianos e histórico de sessão.
- Persistência do estado mais recente da calculadora.
- Testes de domínio, testes de interface automatizados, fluxo de integração contínua e fluxo de distribuição.
- Empacotamento portátil autossuficiente, instalador via Inno Setup e geração de hashes de release.
