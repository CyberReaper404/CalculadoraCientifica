# Versão original do projeto

Esta calculadora nasceu como um projeto de faculdade desenvolvido originalmente em `2021`, com execução em terminal e navegação por menus.

O texto abaixo preserva a proposta da versão inicial, publicada depois no GitHub como registro da evolução da autora.

## Resumo da versão original

Calculadora científica em Python desenvolvida como projeto de faculdade em 2021. Opera pelo terminal com menus interativos para operações básicas, trigonometria, logaritmos e números complexos.

## Como era o projeto

Uma calculadora simples feita em Python para resolver contas do dia a dia e também algumas operações mais avançadas.

Apesar da data de publicação no GitHub ser posterior, o código foi desenvolvido originalmente em 2021 como um trabalho da faculdade.

Na época, a proposta era praticar fundamentos de programação e estruturar o programa com menus, funções e lógica condicional.

## Funcionalidades da versão de terminal

- operações básicas: soma, subtração, multiplicação e divisão
- potência e raízes: potenciação, raiz quadrada e raiz cúbica
- trigonometria: seno, cosseno, tangente e funções inversas em graus
- logaritmos: natural, base 10 e base personalizada
- números complexos: adição, multiplicação, módulo e forma polar

## Execução da versão original

Requisitos:

- Python `3.6` ou superior

Comando:

```bash
python calculadora.py
```

## Estrutura da versão original

O projeto original era composto por um único arquivo, `calculadora.py`, organizado em funções:

- um menu principal que chamava os submenus
- cada categoria de operação com seu próprio fluxo
- uso da biblioteca `math` para os cálculos

## O que esse primeiro projeto representou

Esse foi um dos primeiros projetos da faculdade e teve como foco praticar:

- estruturas de repetição
- condicionais
- funções
- importação de módulos

É um projeto simples, mas importante como marco do começo da trajetória da autora com programação.
