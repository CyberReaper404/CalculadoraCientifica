# CalculadoraCientifica
Calculadora científica em Python desenvolvida como projeto de faculdade em 2021. Opera pelo terminal com menus interativos para operações básicas, trigonometria, logaritmos e números complexos.

🧮 Calculadora Científica em Python

Uma calculadora simples feita em Python para resolver contas do dia a dia e também algumas operações mais avançadas.
📅 Sobre este projeto

Apesar da data de publicação (2026), este código foi desenvolvido originalmente em 2021 como um trabalho da faculdade.

Na época eu ainda não conhecia o Git e não sabia como versionar ou compartilhar projetos. Agora estou organizando e publicando meus projetos antigos para documentar minha evolução como desenvolvedora.
⚙️ Como funciona?

A calculadora funciona pelo terminal (linha de comando). Você escolhe opções numéricas nos menus e vai navegando pelas funcionalidades.
O que tem no menu:

    Operações básicas: soma, subtração, multiplicação, divisão

    Potência e raízes: potenciação, raiz quadrada e cúbica

    Trigonometria: seno, cosseno, tangente e seus inversos (ângulos em graus)

    Logaritmos: natural (ln), base 10 e em qualquer base

    Números complexos: adição, multiplicação, módulo e forma polar

🚀 Como executar

    Tenha o Python instalado no seu computador (versão 3.6 ou superior)

    Baixe ou clone este repositório

    No terminal, navegue até a pasta do projeto

    Execute o comando:

bash

python calculadora.py

📁 Estrutura

O código é um único arquivo (calculadora.py) organizado em funções:

    Um menu principal que chama os submenus

    Cada categoria de operação tem seu próprio menu

    As contas são feitas usando a biblioteca math do próprio Python
